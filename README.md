# DS_PURITY-NAFULA-WEKESA
Data Science assignment analyzing the relationship between Hyperliquid trader behavior and Bitcoin market sentiment (Fear &amp; Greed Index). Includes exploratory data analysis, visualizations, and strategic insights.
# Web3 Trader Behavior Insights

## Project Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and actual trader behavior on the Hyperliquid exchange. The goal is to identify if sentiment shifts act as leading indicators for trader profitability and leverage usage.

## Folder Structure
- `ds_purity_nafula/` (Root)
    - `notebook_1.ipynb`: Data cleaning, merging, and EDA.
    - `csv_files/`: Contains processed data.
    - `outputs/`: Visualizations and charts.
    - `ds_report.pdf`: Final analysis summary.

## Tech Stack
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
